Q1--------------------------------

List: It is an ordered, mutable collection of items.
It will be used when we need to store multiple items and allow addition/modification in items, it can also store duplicates

Tuple: It is an ordered, immutable collection of items.
It will be used when we need to store fixed number of items which needs no modification, but it can store duplicates

Set: It is unordered , mutable collection of items, doesnt allowing duplicates, and provide functionalities of mathematical sets
It will be used for fast searching (O(1)), storing distinct values, like categories, in a data

Dictionary: It is an unordered collection of key-value pairs, with O(1) search of keys
It is used for storing data in form of key and values, also python style of json, so useful in storing data in REST applications

API keys should never be hard-coded because if the code is uploaded to GitHub or shared, anyone can steal and misuse the key.

Recommended approach is to store the API key inside a .env file then use python-dotenv package to load it

Example:

from dotenv import load_dotenv
import os
load_dotenv()
api_key = os.getenv("OPENAI_API_KEY")

from pydantic import BaseModel, Field
from typing import List

class WeatherReport(BaseModel):
    city: str = Field(min_length=1)
    temperature_c: float = Field(ge=-50, le=60)
    humidity: int = Field(ge=0, le=100)
    conditions: List[str]

Valid example:
report = WeatherReport(
    city="Jaipur",
    temperature_c=31.4,
    humidity=42,
    conditions=["sunny", "windy"]
)

Invalid: 
report = WeatherReport(
    city="",
    temperature_c=80,
    humidity=120,
    conditions=["sunny"]
)

import asyncio

async def fetch(url):

        r = await client.get(url, timeout=10)

        r.raise_for_status()

        return r.json()

{
    const [count, setCount] = useState(0);
    return(
        <div>
            <h1>Count :{count}</h1>

import { useEffect, useState } from "react"

export default function SearchableList() {
  const [users, setUsers] = useState([])
  const [loading, setLoading] = useState(true)
  const [error, setError] = useState("")
  const [search, setSearch] = useState("")

  useEffect(() => {
    async function fetchUsers() {
      try {
        const response = await fetch("https://jsonplaceholder.typicode.com/users")

        if (!response.ok) {
          throw new Error("Failed to fetch users")
        }
        const data = await response.json()
        setUsers(data)
      } catch (err) {
        setError(err.message)
      } finally {
        setLoading(false)
      }
    }

    fetchUsers()
  }, [])

  const filteredUsers = users.filter(user => user.name.toLowerCase().includes(search.toLowerCase()))

  if (loading) {
    return <h2>Loading...</h2>
  }

  if (error) {
    return <h2>{error}</h2>
  }

  return (
    <div>
      <input
        type="text"
        placeholder="Search users"
        value={search}
        onChange={(e) => setSearch(e.target.value)}
      />

      {filteredUsers.length === 0 ? (
        <p>No users found</p>
      ) : (
        <ul>
          {filteredUsers.map(user => (
            <li key={user.id}>{user.name}</li>
          ))}
        </ul>
      )}
    </div>
  )
}
